# Pricing_strategy-KS
Python based pricing strategy case study with inventory aware pricing logic, margin guardrails, competitive benchmarking, and final SKU level recommendations.

## Overview
This repository contains the full analysis and pricing logic implemented as part of the pricing strategy case study.

## Contents
- Pricing_Analysis.ipynb — Analysis, pricing logic, and visualizations
- Final_Pricing_Recommendations.csv — Final SKU-level pricing output

## How to Run
Open the Jupyter notebook and run all the cells.
